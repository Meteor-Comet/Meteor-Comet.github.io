# 电气元件认识

在非标自动化上位机开发与电控系统设计中，电气元件的物理选型、工作原理、接线规范以及通信调试是保障整机稳定性的关键。如果上位机软件工程师不了解底层电气接线和传感器原理，在遇到信号抖动、通信丢包或原点回零异常时，往往无从下手。

本文以一套**石墨盘自动装卸整机项目**的真实 BOM 清单为核心，系统性拆解整机电气架构。图文并茂地解析工控机、分布式 IO 总线、精密电机、相机视觉、扫码枪以及防干扰接线配件的工作原理、接线电路与调试要点。

## 一、控制系统与人机交互

控制系统作为设备的“大脑”，不仅需要运行多轴插补算法，还要兼顾高分辨率图像处理与数据库数据上报。

### 1.1 工业 PC (工控机) — 源控电子

![工控机](/img/in-post/electrical-components/control-ipc.png)

#### 1. 工作原理
工控机 (Industrial Personal Computer, IPC) 是一种加固的增强型个人计算机。它与普通家用/商用电脑最大的区别在于**硬件稳定性与扩展性**。它采用宽温设计、无风扇被动散热（通过铝型材鳍片），能有效抵抗电控柜内的电磁干扰、粉尘与振动。
上位机控制架构中，工控机运行 Windows 系统及 C# WPF 主控程序，通过 PCIe 插槽挂载多轴运动控制卡，并通过独立以太网口与机器视觉相机、PLC、远程 IO 进行高频通信。

#### 2. 接线与系统拓扑
```text
             ┌────────────────────────────────────────────────────────┐
             │                源控电子 工控机 (IPC)                    │
             └──────┬─────────────────┬────────────────┬──────────────┘
                    │ (PCIe 插槽)      │ (GigE 网口)     │ (EtherCAT)
                    ▼                 ▼                ▼
             ┌──────────────┐   ┌────────────┐   ┌──────────────┐
             │ 固高运动控制卡│   │ 凌云光相机 │   │ 灵猴远程 IO  │
             └──────────────┘   └────────────┘   └──────────────┘
```

#### 3. 现场调试注意事项
- **网卡配置**：工业 GigE 网口必须配置专用巨帧（Jumbo Packet，建议设置为 9014 字节），并关闭网卡的节能模式，防止视觉传输大数据流时发生丢包崩溃。
- **系统热设计**：本整机电控柜侧板配备了 4 个 **德力西 (DELIXI)** 工业风扇，进风口与排风口需对角布置，确保柜内空气流速，使工控机环境温度处于 40°C 以下。

### 1.2 IO 扩展模块 — 固高 HCB5-1616-DTD01

#### 1. 工作原理
在多轴运动控制中，运动控制卡（插在工控机内）板载的 IO 点数非常有限。为了不占用宝贵的总线周期去读取机箱风扇、指示灯等低频开关信号，使用专用 **IO 扩展模块** 挂在控制卡本地扩展总线上，实现信号的硬件级就近采集。

#### 2. NPN型接线图
```text
  外部 NPN 开关/传感器                         固高 HCB5 模块输入端
  ┌──────────────────────┐                     ┌───────────────────────────┐
  │  +24V 电源线 (棕)     ├─────────────────────►│ V+ (模块 24V 输入端)       │
  │  信号输出线 (黑)      ├─────────────────────►│ INx (数字输入通道)         │
  │  0V 地线 (蓝)         ├──────────┬──────────►│ COM (公共端，接 0V)        │
  └──────────────────────┘          │          └───────────────────────────┘
  外部直流开关电源 0V ────────────────┘
```

#### 3. 调试要点
- **极性判定**：固高的扩展 IO 模块通常提供 NPN (SINK) 与 PNP (SOURCE) 极性切换端子，接线前必须确认传感器类型，接错极性将导致输入指示灯不亮或常亮。

### 1.3 SMEMA 信号转接隔离模块 — LH-SMEMA-SCM-GI-12CH

![SMEMA隔离模块](/img/in-post/electrical-components/control-smema-module.png)

#### 1. 工作原理
SMEMA 标准定义了表面贴装设备之间传送 PCB 载具时的握手规范。主要依靠两路隔离的信号线：
- **Ready (已准备就绪)**：本设备有板/托盘可以送出。
- **Busy (正忙)**：下游设备正忙，无法接收。

非标流水线中，上游设备与下游设备的控制系统通常由不同厂家提供，其直流电源系统绝不能共地（存在电势差）。SMEMA 模块利用内部的**光电耦合器**进行物理隔离，将电信号转换为光信号再还原，确保两端设备即便有数百伏电势差，也不会烧毁内部电路。

#### 2. SMEMA 隔离转换接线原理
```text
   上游设备 (发送端)                             隔离转换模块 (LH)                       本设备 (接收端)
  ┌─────────────────┐                           ┌─────────────────────┐                 ┌───────────────┐
  │ [Ready Out 继电器]│                           │                     │                 │               │
  │  Pin 1 (输出+)  ├──────────────────────────►│ IN+                 │                 │               │
  │  Pin 2 (输出-)  ├──────────────────────────►│ IN-  (内部光耦输入)  │                 │               │
  └─────────────────┘                           │                     │                 │               │
                                                │ OUT+ (内部光耦输出)  ├────────────────►│ PLC / 控制卡 DI│
                                                │ OUT-                ├────────────────►│ 0V GND        │
                                                └─────────────────────┘                 └───────────────┘
```

#### 3. 调试注意
- **常开与常闭**：SMEMA 规范中，Ready 和 Busy 触点在正常传输时通常设计为**常开 (NO)**，接线时需仔细核对线缆 Pin 脚定义，防止逻辑反转导致整线停机。

### 1.4 PLC 可编程逻辑控制器

![PLC 可编程逻辑控制器](/img/in-post/electrical-components/control-plc.png)

#### 1. 工作原理
PLC（Programmable Logic Controller，可编程逻辑控制器）是工业自动化的核心控制设备。它采用**扫描周期**（通常为 1~10ms）循环执行用户编写的梯形图或结构化文本程序，根据输入状态计算输出结果，并驱动执行元件动作。

PLC 与上位机工控机的协作分工如下：
- **工控机（上位机）**：负责运动轨迹规划、视觉算法、数据库对接、用户界面 (HMI/WPF)。
- **PLC（下位机）**：负责设备安全联锁、气阀顺序控制、传感器信号采集与工站级状态机逻辑。

#### 2. PLC 典型接线图（西门子 S7-1200 / 三菱 FX5U）
```text
  ┌────────────────────────────────────────────────────────┐
  │                   PLC 主机                              │
  │  电源: L(AC220V) ──► 内部开关电源 ──► 24V DC 传感器供电 │
  │                                                         │
  │  数字输入 DI:                                           │
  │    I0.0 ──────── 近接传感器（棕 24V, 蓝 0V, 黑 Signal） │
  │    I0.1 ──────── 急停按钮（NC 型，常闭接入）             │
  │    I0.2 ──────── 气缸磁开关 (2线制，接 DI 和 0V)        │
  │                                                         │
  │  数字输出 DO:                                           │
  │    Q0.0 ──────── 电磁阀线圈 (+24V，负载，0V)            │
  │    Q0.1 ──────── 继电器线圈（驱动大电流执行器）          │
  │                                                         │
  │  通信:                                                  │
  │    PROFINET / EtherNet/IP ──► 上位机工控机              │
  └────────────────────────────────────────────────────────┘
```

#### 3. 上位机与 PLC 通信要点
- **通信协议**：在非标上位机中，常用 **Modbus TCP**（通用性强）、**OPC UA**（西门子/倍福）或 **三菱 MC 协议**（三菱 FX5U）。
- **心跳机制**：上位机与 PLC 之间须建立周期性心跳检测（如每 100ms 写一次寄存器），PLC 侧若超时未收到心跳，自动触发急停逻辑，保障安全。
- **握手协议**：设计"工位完成信号"与"允许下一步"信号的双向握手，防止并发竞态条件。

### 1.5 运动控制卡

![运动控制卡](/img/in-post/electrical-components/control-motion-card.png)

#### 1. 工作原理
运动控制卡是安装在工控机 PCIe 插槽内的专用扩展卡，负责高精度多轴联动运动控制。它通过 **DSP/FPGA** 实时处理插补算法，以 **EtherCAT** 或 **脉冲+方向（Pulse/Dir）** 两种方式驱动伺服/步进电机。

运动控制卡相比 PLC 的优势：
- **实时性更高**：FPGA 级别的插补，轨迹精度达到亚微米级。
- **编程更灵活**：可用 C#/C++ SDK 直接调用，无需专用编程软件，与 WPF 上位机深度融合。
- **多轴联动**：支持 X/Y/Z/R 四轴及以上的插补联动，适合复杂搬运与焊接轨迹。

#### 2. 脉冲型接线方式（固高 GT-400 / 雷赛 MC8040）
```text
  ┌────────────────────────────────────────────────────────┐
  │             运动控制卡 (PCIe 安装于工控机)               │
  │                                                         │
  │  轴 1 输出:                                             │
  │    PULSE+ / PULSE- ───► 伺服驱动器 PUL+ / PUL- (脉冲)  │
  │    DIR+   / DIR-   ───► 伺服驱动器 DIR+ / DIR- (方向)   │
  │    ENA+   / ENA-   ───► 伺服驱动器 ENA+ / ENA- (使能)   │
  │                                                         │
  │  限位与原点输入 (差分或单端):                            │
  │    LMT+  ─────────────► PM-Y45 正限位信号 (INx)         │
  │    LMT-  ─────────────► PM-Y45 负限位信号 (INx)         │
  │    HOME  ─────────────► PM-Y45 原点信号 (INx)           │
  └────────────────────────────────────────────────────────┘
```

#### 3. 调试要点
- **软限位 + 硬限位双重保护**：运动控制卡软件侧设置软限位坐标，硬件侧连接槽型传感器（PM-Y45），二者形成双重防止过冲保护。
- **回零策略**：常用的回零模式为"先高速粗找原点信号，减速后低速精找"，精度在 ±1 个编码器脉冲以内。

### 1.6 工业触摸屏 (HMI)

![工业触摸屏 HMI](/img/in-post/electrical-components/hmi-touchscreen.png)

#### 1. 工作原理
HMI（Human-Machine Interface，人机界面）是现场操作员与设备控制系统交互的界面终端。工业级触摸屏具备防尘、防油、宽温、抗振动设计，IP65 防护等级，可在恶劣工业环境中长期稳定运行。

在不含工控机的纯 PLC 系统中，HMI 通过 PROFINET/Modbus 连接 PLC，负责参数配置、报警显示与手动操控。在含工控机的系统中，HMI 可退化为纯显示终端，或通过以太网与上位机 WPF 程序联动。

#### 2. HMI 通信接线图
```text
  [威纶通 MT8000 / 昆仑通态 TPC7072]
  ┌────────────────────────────────────┐
  │  电源: 24V DC ──► 内部稳压         │
  │  RS-485 (COM1):                    │
  │    A+ ──────────► PLC RS-485 A     │
  │    B- ──────────► PLC RS-485 B     │
  │  以太网 (Ethernet):                 │
  │    RJ45 ────────► 工控机 / 交换机   │
  └────────────────────────────────────┘
```

#### 3. 选型建议
- **现场操作面板**：推荐选用 7~10 英寸触摸屏（威纶通 MT8071/MT8102），满足参数设置与手动操控需求。
- **主监控大屏**：可与工控机 WPF 软件界面配合，使用 21~27 英寸工业显示器，通过 HDMI/DP 直连工控机显卡，无需额外 HMI 协议开发。

## 二、总线分布式 IO 与通信架构

在中大型自动化整机中，为了避免数百米长电缆的复杂布线，总线架构是必由之路。

### 2.1 远程总线 IO 模块 — 灵猴 LHS3 系列 EtherCAT 模块

#### 1. 工作原理
远程 IO 模块是现场传感器与主控系统之间的桥梁。它通过高速 **EtherCAT 工业以太网** 与工控机主站进行周期性过程数据通信（通信周期通常为 1ms 或 500μs）。模块直接安装在工站下方，实现“就近接线、网线总线传输”。

#### 2. 本整机项目中的模块分布与连接
BOM 清单中一共应用了 3 种规格的灵猴总线模块，分布于整机 7 个大部件中：
- **LHS3-3200-XET1 (32路数字输入)**：
  - **安装位置**：交错上料组件 (1 PCS)。
  - **接入元件**：集中接入该站的 17 个 EX-14A、4 个 EX-13EA、4 个 PM-Y45 限位原点以及 4 个气缸磁开关。
- **LHS3-1616-XET1 (16输入/16输出)**：
  - **安装位置**：中转搬运组件 (1 PCS)、下料搬运组件 (1 PCS)。
  - **接入元件**：输入接 PM-Y45，输出接真空一体阀。
- **LHS3-2408-XET1 (24输入/8输出)**：
  - **安装位置**：下料翻转组件 (1 PCS)、流线1 (1 PCS)、流线2 (1 PCS)、流线3 (1 PCS)。
  - **接入元件**：输入接流线 EX-13 传感器，输出接阀岛。

#### 3. 总线网线“手拉手”串联调试
所有的 LHS3 模块带有两个 RJ45 总线网口（ECAT IN 和 ECAT OUT），调试时必须严格遵循手拉手（Daisy Chain）串行连接：
```text
  [工控机主站网口] ──► [LHS3-3200 IN] ──► [LHS3-3200 OUT] ──► [LHS3-1616 IN] ──► [LHS3-1616 OUT] ──► ...
```
- **布线注意**：EtherCAT 对网线要求较高，强干扰区必须使用超五类（CAT5e）或六类（CAT6）带屏蔽的双绞线（SFTP），且屏蔽层必须在电控柜端单端接地，防止伺服电机高频谐波干扰通信。

### 3.3 槽型光电传感器 — 松下 PM-Y45 / PM-T45

#### 1. 工作原理与定位精度

### 3.6 近接传感器（电感式接近开关）

![近接传感器](/img/in-post/electrical-components/sensor-proximity.png)

#### 1. 工作原理
电感式接近传感器（Inductive Proximity Sensor）利用**高频振荡 LC 电路**产生交变磁场。当金属物体进入感应范围时，在金属表面产生涡流，消耗 LC 回路的振荡能量，导致振幅衰减，内部检波电路检测此变化并触发输出。

与光电传感器相比：
- **优点**：不受油污、粉尘、水雾遮挡影响；无需对准，安装宽松。
- **局限**：**只能检测金属**（铁磁体感应距离最远，铝/铜等有色金属感应距离仅约 1/3）。

常见规格：外壳直径 M8/M12/M18/M30；感应距离 1.5~15mm；输出方式 NPN/PNP；连接方式 2线制/3线制/4线制。

#### 2. NPN 型 3 线制接线图
```text
  近接传感器 (NPN 型)                      控制模块输入端
  ┌──────────────────────┐                ┌───────────────────────────┐
  │  棕色线 (Brown)  ────┼───────────────►│ V+ (+24V DC)              │
  │  蓝色线 (Blue)   ────┼───────────────►│ V- (0V GND)               │
  │  黑色线 (Black)  ────┼───────────────►│ INx (信号输入)             │
  └──────────────────────┘                └───────────────────────────┘
  NPN 型输出低电平，信号线在检测到金属时拉低至 0V
```

#### 3. 应用场景
- **机械手末端金属工件到位检测**：检测吸嘴吸附的金属载盘是否到位。
- **气缸行程端检测**：在气缸侧面固定，检测活塞杆是否完全伸出/缩回（适用于铁活塞杆）。
- **旋转体转速检测（齿轮盘计数）**：对准旋转齿轮盘齿顶，每过一个齿产生一个脉冲，用于低精度转速测量。

### 3.7 磁性传感器（气缸磁开关）

![气缸磁性传感器](/img/in-post/electrical-components/sensor-magnetic.png)

#### 1. 工作原理
气缸磁开关（Cylinder Magnetic Reed Switch）安装于气缸外壁的 T 型槽或 C 型槽中，内部采用**干簧管（Reed Switch）**或**霍尔效应传感器**。当气缸活塞（内嵌永磁体）运动到传感器位置时，永磁场触发干簧管闭合（或霍尔元件产生输出），指示气缸达到行程终点。

| 类型 | 典型型号 | 原理 | 特点 |
|:---|:---|:---|:---|
| 干簧管型 | SMC D-M9N | 磁场使两片弹簧触片接触 | 成本低，寿命约 1000万次 |
| 霍尔效应型 | SMC D-A93 | 半导体霍尔效应输出 | 寿命长，响应快，不怕震动 |
| 固态型（PNP/NPN）| 费斯托 SME | 半导体固态输出 | 无接触，IP67 防护 |

#### 2. 2 线制接线图
```text
  气缸磁开关 (2线制，干簧管型)
  ┌──────────────────────┐
  │  棕色线 ─────────────┼───► PLC/IO 模块 INx（数字输入通道）
  │  蓝色线 ─────────────┼───► 0V GND
  └──────────────────────┘
  注意：2线制磁开关内阻约数百欧，须经输入通道内部负载才能正常工作
```

#### 3. 应用场景与调试
- **气缸双端行程确认**：每个气缸通常安装两个磁开关（伸出到位 + 缩回到位），用于工序联锁确认。
- **安装位置调整**：沿 T 型槽滑动磁开关至指示灯亮起的位置后锁紧，调试时需确认磁开关与气缸活塞端点位置对齐。

### 3.8 压力传感器

![数字压力传感器](/img/in-post/electrical-components/sensor-pressure.png)

#### 1. 工作原理
压力传感器将介质（气体/液体）的压力转换为电信号输出。在自动化设备中最常用的是**数字式压力传感器**，内置压敏电阻桥（Wheatstone Bridge）和 A/D 转换芯片，前面板数字显示实时压力值，并提供开关量输出（高/低压报警）。

常见输出方式：
- **开关量 (NPN/PNP)**：设定上下限阈值，超出时触发输出，用于"真空吸附成功"判断。
- **模拟量 (4~20mA / 0~10V)**：连接 PLC/工控机模拟量输入端，实时监控压力曲线。

#### 2. 真空检测接线图（开关量型）
```text
  [数字压力传感器 松下 DP-100 / SMC ZSE30]
  ┌──────────────────────┐
  │  棕色线 ──────────────┼───► +24V
  │  蓝色线 ──────────────┼───► 0V GND
  │  黑色线 (OUT1) ───────┼───► PLC/IO INx（真空到达判断）
  │  白色线 (OUT2) ───────┼───► PLC/IO INx（真空泄漏报警）
  └──────────────────────┘
```

#### 3. 应用场景
- **吸盘真空检测**：设定阈值为 -60kPa，真空达到则判定吸附成功，允许搬运轴运动。
- **气路供气压力监控**：将主气路压力接入压力传感器，低于 0.4MPa 时触发声光报警。

### 3.9 温度传感器 (PT100)

![PT100 温度传感器](/img/in-post/electrical-components/sensor-temperature.png)

#### 1. 工作原理
PT100 铂电阻（RTD）利用铂金属阻值随温度线性变化的特性：在 0°C 时电阻值为 100Ω，温度每升高 1°C，阻值约增加 0.385Ω。精度远高于热电偶（Thermocouple）。

|:---|:---|:---|:---|

| NTC 热敏电阻 | -50~150°C | ±0.5°C | 2线 |

#### 2. 3 线制 PT100 接线图（消除引线电阻误差）

  └──────────────────────┘                  └───────────────────────────┘

#### 3. 应用场景
- **加热台温控**：控制热压贴合机的加热台温度（如 180°C±2°C）。
- **电机/驱动器过热保护**：安装在驱动器散热片上，超过 80°C 时触发报警停机。

### 3.10 光纤传感器

![光纤传感器](/img/in-post/electrical-components/sensor-fiber.png)

#### 1. 工作原理
光纤传感器（Fiber Optic Sensor）由**放大器单元（Amplifier Unit）**和**光纤探头**两部分组成。放大器内置光源（LED/激光）和接收电路，光纤仅作为导光介质，将光引导至机构末端极小的检测点。

**核心优势**：光纤头体积极小（最小直径 φ1mm），可以安装在普通传感器无法安装的极狭小空间内，且光纤本身不带电，适用于强电磁干扰或高温区域。

检测方式：对射型（发射与接收光纤对置，检测遮光）；反射型（发射/接收光纤同侧，检测反射光）。

#### 2. 接线图（放大器单元）
```text
  松下 FX-500 / 基恩士 FS-N 光纤放大器
  ┌──────────────────────┐
  │  棕色线 ──────────────┼───► +24V
  │  蓝色线 ──────────────┼───► 0V
  │  黑色线 (OUT) ────────┼───► PLC/IO INx
  └──────────────────────┘
  光纤探头接入放大器两侧的光纤插口（ST型或SMA型接口）
```

#### 3. 应用场景
- **高速流水线微小工件检测**：检测 φ0.5mm 的细小零件。
- **透明物体检测**：通过激光型光纤传感器检测玻璃、透明薄膜是否存在。

### 3.12 编码器

![编码器](/img/in-post/electrical-components/sensor-encoder.png)

#### 1. 工作原理
编码器（Encoder）将轴的旋转位置或直线位移转换为数字脉冲信号，是运动控制系统中位置反馈的核心元件。

| 类型 | 特点 | 典型用途 |
|:---|:---|:---|
| **增量型编码器** | 断电丢失位置，需每次上电回零 | 伺服电机位置反馈 |
| **绝对值编码器** | 断电保持位置，单圈/多圈分辨率高 | 机器人关节、旋转轴 |
| **光栅尺（线性编码器）** | 直接测量直线位移，精度达 0.1μm | 直线电机、精密工作台 |

#### 2. 编码器输出方式与接线
```text
  增量型编码器（欧姆龙 E6B2-CWZ6C）
  ┌──────────────────────────────────────────────────────────┐
  │  棕(+V) ──► +24V        蓝(0V) ──► GND                  │
  │  A相(黑) ──► 控制卡 A相输入（正交计数主相）              │
  │  B相(白) ──► 控制卡 B相输入（正交计数辨向相）            │
  │  Z相(橙) ──► 控制卡 Z相输入（每圈一个零点脉冲）          │
  └──────────────────────────────────────────────────────────┘
  A/B相正交信号可 4 倍频，1000ppr 可提升至 4000 计数/圈
```

#### 3. 应用场景
- **输送带线速度测量**：将编码器安装在输送带辊轴，实时测量线速度与传送距离，用于变速控制。
- **旋转角度检测**：在旋转工位（R 轴）上安装编码器，精确控制旋转角度（如 90°、180° 定位）。

## 四、动力与气动执行机构

动力单元是上位机输出指令的终点，负责驱动大跨度机械手和翻转机构。

### 4.1 精密直线电机 — 灵猴系列

![直线电机](/img/in-post/electrical-components/actuator-linear-motor.png)

#### 1. 工作原理
直线电机（Linear Motor）是一种将电能直接转换为直线往复运动的传动装置。它取消了伺服电机+滚珠丝杠/同步带等机械转换机构：
- **定子**：铺设在底座上的强力永磁轨磁条。
- **动子**：内部带有三相电磁绕组的滑块。
由于非接触传动，系统机械间隙为 0，最大加速度可达几 G。滑台上配有高精度**光栅尺**（反馈分辨率通常为 0.1μm），实现极其闭环的位置反馈。

#### 2. 直线驱动控制系统结构
```text
  [工控机/控制卡] ──── 总线指令 ───► [直线电机驱动器] ◄─── (光栅尺读数) ──┐
                                       │ (驱动电流 U/V/W)             │
                                       ▼                            │
                                ┌─────────────┐                     │
                                │   动子滑块   ├─────────────────────┘
                                └─────────────┘
                                [定子磁轨磁条]
```

#### 3. 实战应用位置
- **交错上料组件 (2 PCS)**：Y1、Y2 直线电机高速交错运行，交替完成石墨盘的平稳喂料。
- **定位移栽与下料搬运 (Y轴)**：采用灵猴直线电机，保障高频循环往复搬运时不产生物理磨损，维护周期长。

---

### 4.2 交流伺服电机 — 汇川系列

#### 1. 工作原理
交流伺服系统采用三相正弦波交流电驱动电机转子。电机尾部带有高分辨率绝对值编码器（如 23-bit），将实际旋转位置和转速反馈给伺服驱动器，实现高响应（通常频响在 3.0kHz 以上）的位置闭环调节。

#### 2. 项目大部件轴分布表
整机多处中行程和辅助旋转轴选用了**汇川 (INOVANCE)** 伺服系统：
- **上料摆正 Y1 轴 (1 PCS)**：对进入工站的盘体实现精细的前后对齐微调。
- **中转搬运 X/Z 轴 (2 PCS)**：控制拾取搬运爪的高速移动与垂直升降。
- **下料搬运 X/Z/R 轴**：控制搬运轴与旋转 R 轴。
- **下料翻转轴 (1 PCS)**：将装满物料的石墨盘整体翻转 180 度。

---

### 4.3 智能阀岛与真空一体阀

![阀岛](/img/in-post/electrical-components/actuator-solenoid-valve.png)

#### 1. 工作原理
- **总线式阀岛 (EMC 亿太诺)**：将原本分散的电磁阀集中集成在一块汇流板上，共用进气口、排气口和总线接口。
- **真空一体阀**：利用文丘里（Venturi）效应，通过压缩空气高速穿过狭窄管道在内部产生负压，直接吸起工件。它集成了吸气阀、吹气破坏阀和真空数字压力表，响应延迟可控制在 5ms 以内。

#### 2. 真空一体阀气动与电气控制逻辑
```text
  [电控输出 24V_Solenoid A] ──► 真空阀通电 ──► 吸盘产生真空吸附
  [电控输出 24V_Solenoid B] ──► 破坏阀通电 ──► 吹气强行剥离工件
  [真空压力开关信号] ─────────► 反馈给 PLC ──► 大于 -60kPa 判定吸附成功
```

#### 3. 实战分布
- 流线 1 与流线 2 分别配置了 1 套 **亿太诺 (EMC)** 智能阀岛控制各阻挡顶升气缸。
- 中转搬运 (4个)、定位移栽 (2个)、下料翻转 (2个) 机构末端各配备了多套真空一体阀驱动多联吸盘。

---

### 4.4 气缸

![气缸](/img/in-post/electrical-components/actuator-cylinder.png)

#### 1. 工作原理
气缸（Pneumatic Cylinder）是将压缩空气的压力能转换为直线往复运动机械能的执行器。在非标自动化设备中，气缸凭借**动作快（响应时间 < 50ms）、价格低廉、结构简单、维护方便**等优点，大量用于短行程的夹紧、顶升、阻挡和推料等动作。

常见气缸类型：

| 类型 | 特点 | 典型应用 |
|:---|:---|:---|
| 标准双作用气缸 | 进/排气双向控制，可控速度 | 推料、顶升、阻挡 |
| 薄型气缸（SMC CQ2）| 扁平结构，安装空间小 | 机械手夹爪、压板 |
| 无杆气缸 | 行程长，占用轴向空间小 | 长行程搬运导向 |
| 旋转气缸 | 输出旋转运动（90°/180°）| 翻转、拨料动作 |
| 锁紧气缸 | 断气自锁，防止掉落 | 垂直轴夹紧、工件锁固 |

#### 2. 单电控电磁阀驱动双作用气缸接线图
```text
  电控输出端 (PLC Q0.x)                   电磁阀 (SMC SY系列)
  ┌────────────────────┐                  ┌──────────────────────────┐
  │  DO 输出 +24V ─────┼─────────────────►│ SOL+ 线圈 A 正端          │
  │  0V GND       ─────┼─────────────────►│ SOL- 线圈 A 负端          │
  └────────────────────┘                  └────────────┬─────────────┘
                                                       │ (气路切换)
                                           ┌───────────┴───────────────┐
                                           │ A口(进气) → 气缸后腔(伸出)│
                                           │ B口(排气) ← 气缸前腔(排气)│
                                           └───────────────────────────┘
```

#### 3. 调速与缓冲调试
- **速度控制**：在气缸进排气管路上安装**节流阀（速度控制阀）**，调节进气（进口节流）或排气（出口节流）流量，控制气缸运动速度。**推荐使用排气节流**，可减少气缸蠕动现象。
- **缓冲调整**：气缸内置液压缓冲器，末端带有调节螺钉，顺时针旋转增大缓冲（降低末端冲击），高速应用时必须调整，防止撞坏机构。

### 4.5 步进电机与驱动器

![步进电机](/img/in-post/electrical-components/actuator-stepper-motor.png)

#### 1. 工作原理
步进电机（Stepper Motor）通过驱动器将脉冲电信号转换为固定角度的机械旋转，每接收一个脉冲旋转固定步距角（通常 1.8°/步，细分后可达 0.036°/步）。步进电机是一种**开环控制**方式，无需编码器反馈，价格远低于伺服系统，适合精度要求中等（重复定位精度 ±0.05mm 级别）的场合。

步进电机 vs 伺服电机对比：
| 对比项 | 步进电机 | 伺服电机 |
|:---|:---|:---|
| 控制方式 | 开环（无反馈） | 闭环（编码器反馈） |
| 失步风险 | 过载时会失步 | 闭环自动补偿 |
| 价格 | 低（驱动器 + 电机 约 200~500 元）| 高（驱动器 + 电机 约 2000~8000 元）|
| 适用精度 | ±0.05mm 级（细分细分后）| ±0.001mm 级 |

#### 2. 接线图（脉冲方向型驱动器 DM542）
```text
  运动控制卡 / PLC 高速输出                 步进驱动器 (DM542)
  ┌────────────────────┐                  ┌──────────────────────────┐
  │  PUL+ ─────────────┼─────────────────►│ PUL+  (脉冲正)            │
  │  PUL- ─────────────┼─────────────────►│ PUL-  (脉冲负)            │
  │  DIR+ ─────────────┼─────────────────►│ DIR+  (方向正)            │
  │  DIR- ─────────────┼─────────────────►│ DIR-  (方向负)            │
  │  ENA+ ─────────────┼─────────────────►│ ENA+  (使能正)            │
  │  ENA- ─────────────┼─────────────────►│ ENA-  (使能负)            │
  └────────────────────┘                  │                          │
                                          │ A+ A- B+ B- ─────────────►  两相步进电机绕组
                                          │ VCC (24~50V DC 电源)      │
                                          └──────────────────────────┘
```

#### 3. 细分设置与调试
- **细分拨码开关**：DM542 等驱动器通过板载 DIP 拨码开关设定细分数（2/4/8/16/32/64/128/256）。细分越高，单步角度越小，运行越平滑，但需要驱动器输出更高频率的脉冲。
- **电流设定**：根据电机额定电流设定驱动器电流，过大则电机过热，过小则力矩不足导致失步。

### 4.6 变频器 (VFD)

![变频器](/img/in-post/electrical-components/actuator-inverter.png)

#### 1. 工作原理
变频器（Variable Frequency Drive, VFD）通过改变输出交流电的**频率**来控制三相异步电机的转速。核心原理：`转速 n = (60 × 频率 f) / 极对数 p`，改变输出频率即可无级调速。内部由整流器（AC→DC）、滤波电容（直流母线）和逆变器（DC→AC，IGBT 桥）三部分组成。

变频器在非标自动化中的典型应用：输送带速度调节、风机泵类调速节能、大扭矩低速运行（拉伸机、缠绕机）。

#### 2. 外部接线图（汇川 MD310 / 台达 VFD-B）
```text
  三相电源 (RST)
      │
  ┌───▼──────────────────────────────────────────────────────────┐
  │                   变频器主机                                  │
  │  R / S / T ──────── 主电路输入（三相 380V AC）                │
  │  U / V / W ──────── 主电路输出 ──► 三相异步电机               │
  │                                                               │
  │  控制端子:                                                    │
  │  DI1 ───────── 正转启动信号 (PLC DO 输出)                     │
  │  DI2 ───────── 反转信号 (PLC DO 输出)                         │
  │  AI1 ───────── 模拟量频率给定 (0~10V, 对应 0~50Hz)             │
  │  DO1 ───────── 故障报警输出 ──► PLC DI                        │
  │  RS-485 ──────── Modbus RTU 通信（上位机远程控制频率）         │
  └────────────────────────────────────────────────────────────┘
```

#### 3. 调试参数设置要点
- **加减速时间**：根据负载惯量合理设置加速时间（P0.17）与减速时间（P0.18），过短会触发过流/过压保护。
- **载波频率**：载波频率越高，电机噪音越小，但 IGBT 发热增加；一般设为 4~8kHz。
- **自动转矩补偿**：启用矢量控制（V/F 控制或无速度传感器矢量）以获得低频段足够的力矩。

## 五、配电与保护元件

在非标自动化设备的电控柜设计中，配电与保护元件是电气安全的最后防线。合理的断路器分级保护、接触器容量选型与继电器隔离方案，是保障设备不发生电气事故的基础。

### 5.1 开关电源

![开关电源](/img/in-post/electrical-components/power-supply.png)

#### 1. 工作原理
开关电源（Switching Mode Power Supply, SMPS）将工频 AC 220V 输入通过高频开关变换（整流→高频斩波→变压→整流滤波）转换为稳定的直流输出，效率高达 85~95%，体积远小于线性电源。

工业设备中 24V DC 开关电源是使用最广泛的控制电源，为 PLC、传感器、电磁阀线圈、指示灯等所有 24V 负载供电。

#### 2. 接线图（明纬 SDR-120-24）
```text
  AC 输入                          DC 输出
  ┌──────────────────────────────────────────────────────────┐
  │  L (AC 220V 相线) ──────────►                            │
  │  N (零线)         ──────────► 内部稳压/开关电路          │
  │  PE (保护地)      ──────────►                    +V ────►│ 24V DC 正极
  │                                                   -V ────►│ 24V DC 负极 (GND)
  └──────────────────────────────────────────────────────────┘
```

#### 3. 选型与注意事项
- **容量选型**：将所有 24V 负载电流求和后乘以 1.2 余量系数，再选最近档位的电源。例如：传感器总电流 1.5A + 电磁阀 3A = 4.5A × 1.2 = 5.4A → 选 **10A (SDR-120-24)**。
- **多路分配**：中大型设备建议按功能分区使用多个独立电源：控制信号电源（传感器/PLC IO）与执行元件电源（阀线圈/继电器）物理分离，防止执行元件瞬态冲击干扰控制信号。

---

### 5.2 24V 电源分配模块

![24V 电源分配模块](/img/in-post/electrical-components/power-distribution-module.png)

#### 1. 工作原理
24V 电源分配模块（Power Distribution Terminal, PDT）将一路 24V DC 输入分配为多路独立输出，每路配备独立的小型熔断器保护。当某路负载短路时，仅该路熔断器熔断，不影响其他路，极大提高了故障隔离能力。

典型产品：菲尼克斯 MINI MCR 系列 / 魏德米勒 WPD 系列。

#### 2. 接线示意
```text
  开关电源 +24V ──────────────────► 分配模块 输入+
  开关电源 GND  ──────────────────► 分配模块 输入-

  分配模块 OUT1+ ──► 传感器组1 (含熔断器 0.5A)
  分配模块 OUT2+ ──► 传感器组2 (含熔断器 0.5A)
  分配模块 OUT3+ ──► 电磁阀组  (含熔断器 2A)
  分配模块 OUT4+ ──► 指示灯    (含熔断器 0.5A)
```

---

### 5.3 断路器 / 空气开关

![断路器](/img/in-post/electrical-components/control-circuit-breaker.png)

#### 1. 工作原理
断路器（Circuit Breaker）是电路过载/短路保护的核心元件，集过流脱扣（热磁型）与短路瞬时脱扣于一体。当电流超过额定值时，内部双金属片弯曲（热效应）触发脱扣机构；短路时，电磁力瞬间触发脱扣（响应时间 < 1ms）。

在电控柜中，断路器实现**分级保护**：
```text
  总进线断路器 (63A) → 开关电源断路器 (10A) → 各支路断路器 (2~6A)
                      → 伺服驱动器断路器 (16A)
                      → 控制回路断路器 (4A)
```

#### 2. 接线
```text
  进线 (相线 L) ──► 断路器 上端 1/3/5 ──► 出线 下端 2/4/6 ──► 负载
  进线 (零线 N) ──────────────────────────────────────────────► 负载
```

#### 3. 选型要点
- **额定电流**：按负载最大电流的 1.25 倍选型，切勿过大（失去保护意义）或过小（频繁误跳）。
- **断路器与熔断器的区别**：断路器可以手动复位重用，熔断器（保险丝）熔断后需更换，断路器更适合工业场合的频繁开关操作。

---

### 5.4 接触器

![接触器](/img/in-post/electrical-components/control-contactor.png)

#### 1. 工作原理
接触器（Contactor）是一种利用电磁铁吸合原理控制大电流负载通断的开关元件。线圈通电→铁芯产生磁场→吸合衔铁→主触点闭合，实现"小电流控制大电流"。

接触器与继电器的核心区别：
- **接触器**：专为频繁通断大功率负载（电机、加热丝）设计，主触点额定电流 9~630A，带弧焰隔离腔（灭弧）。
- **继电器**：用于小电流信号控制与隔离，额定电流通常 ≤ 10A。

#### 2. 典型电机控制接线图
```text
  PLC DO ──► 接触器线圈 A1/A2 (线圈额定: 24V DC 或 220V AC)
                  ↓ (吸合后)
  三相电源 L1/L2/L3 ──► 接触器主触点 1/3/5 ──► 2/4/6 ──► 电机
```

#### 3. 辅助触点应用
接触器自带若干常开(NO)与常闭(NC)辅助触点，常用于：
- **自锁**（起保停控制回路）：常开辅助触点并联在启动按钮两端，维持线圈通电。
- **互锁**（正反转防相冲）：正转接触器的常闭触点串联在反转线圈回路中，防止同时吸合造成相间短路。

---

### 5.5 电磁继电器

![电磁继电器](/img/in-post/electrical-components/control-relay.png)

#### 1. 工作原理
电磁继电器（Electromagnetic Relay）通过小电流驱动线圈产生磁场，吸合铁芯带动触点组切换，实现电气隔离控制。在工业控制系统中，继电器主要作为：
- **电气隔离**：PLC 24V 输出隔离驱动 220V AC 设备（风扇、指示灯、警报器）。
- **电流放大**：PLC 输出电流仅几十毫安，通过继电器触点驱动安培级负载。
- **逻辑控制**：多路继电器触点的串并联实现简单逻辑（用于无 PLC 的简单控制电路）。

#### 2. 继电器插座接线图（欧姆龙 MY2N 8脚型）
```text
  插座引脚图 (从底部看)
  ┌──────────────────────────────────────────────────────────┐
  │   1 ────── 线圈+ (接 PLC DO 或 24V)                       │
  │   2 ────── 线圈- (接 0V GND)                              │
  │   3 ────── 公共端 COM1                                    │
  │   4 ────── 常开 NO1 (线圈通电后与 COM1 通)                │
  │   5 ────── 常闭 NC1 (线圈通电前与 COM1 通)                │
  │   6 ────── 公共端 COM2                                    │
  │   7 ────── 常开 NO2                                       │
  │   8 ────── 常闭 NC2                                       │
  └──────────────────────────────────────────────────────────┘
```

#### 3. 继电器板与继电器模块
在高密度 IO 系统中，常使用**继电器模块组（Relay Module Board）**，将 8/16 路继电器整合在导轨式模块上，PLC 输出直接驱动，输出侧接大功率执行器，接线整洁、维护方便。

### 5.6 固态继电器 (SSR)

![固态继电器](/img/in-post/electrical-components/control-ssr.png)

#### 1. 工作原理
固态继电器（Solid State Relay, SSR）是使用半导体器件（SCR 可控硅或 TRIAC 双向晶闸管）代替机械触点的电子继电器。

| 对比项 | 电磁继电器 | 固态继电器 |
|:---|:---|:---|
| 开关速度 | 毫秒级 | 微秒级（过零触发） |
| 寿命 | 约 100 万次机械寿命 | 半导体寿命，近乎无限 |
| 触点声响 | 有嗒嗒声 | 完全无声 |
| 控制电压 | 通常 24V DC 线圈 | 3~32V DC 或光耦输入 |
| 发热 | 较少 | 需要散热片（导通压降约 1.5V）|

#### 2. AC 型 SSR 接线图（驱动加热丝）
```text
  控制端 (输入):                   负载端 (输出, AC 型):
  PLC DO (+24V) ──► 3/4 脚  (+) │    1/2 脚 (AC 电源端)  ──────► 加热丝 AC 220V
  GND           ──► 3/4 脚  (-) │                   ───────────► AC 零线 N
```

#### 3. 应用场景
- **温度控制 (PID + SSR)**：SSR 配合 PID 温控器，通过 PWM 周期通断控制加热丝功率，实现精密温度控制（精度 ±0.5°C）。
- **无触点频繁开关**：在每分钟开关次数超过 100 次的场合，SSR 的无触点优势明显，可替代继电器延长寿命。

## 六、人机交互与报警元件

现场操作人员通过指示灯、报警灯和按钮直接与设备进行状态交互，是设备安全使用和操作规范的基础。

### 6.1 指示灯（信号灯）

![指示灯](/img/in-post/electrical-components/indicator-light.png)

#### 1. 工作原理
工业指示灯（Signal Lamp / Pilot Light）通过 LED 或传统灯泡的亮/灭/闪烁状态，直观反映设备或某个工站的运行状态。常见颜色及含义（IEC 60073 标准）：

| 颜色 | 含义 | 典型用途 |
|:---|:---|:---|
| **绿色** | 正常/运行中 | 设备正常运行指示 |
| **红色** | 故障/停止 | 报警、急停状态指示 |
| **黄色/橙色** | 警告/注意 | 低物料、维护提醒 |
| **蓝色** | 强制/手动 | 手动调试模式指示 |
| **白色** | 电源/中性 | 电源上电指示 |

#### 2. 接线图（22mm 面板安装型 LED 指示灯）
```text
  PLC DO 输出 ─────► 指示灯 A1 (+)
  GND (0V) ─────────► 指示灯 A2 (-)
  ⚠ LED 型指示灯工作电压通常为 24V DC，无需限流电阻
```

---

### 6.2 蜂鸣器与三色报警灯

![蜂鸣器三色报警灯](/img/in-post/electrical-components/buzzer-stack-light.png)

#### 1. 工作原理
**三色报警灯（Stack Light / Signal Tower）**将三色（红/黄/绿）LED 灯柱堆叠于同一灯座，通过不同组合和闪烁模式向现场操作员和远处的生产管理人员传递设备状态。顶部通常集成蜂鸣器，发出声音报警。

常用报警逻辑定义：
```text
  绿灯常亮  = 设备正常运行中
  黄灯闪烁  = 警告（物料不足/需人工干预）
  红灯常亮  = 故障停机（需要复位才能继续）
  红灯闪烁  = 紧急故障（急停/安全门打开）
  蜂鸣器鸣叫 = 随报警灯联动，引起操作员注意
```

#### 2. 接线图（DC 24V 型）
```text
  PLC DO 输出1 ──► 绿灯 A1    (GND A2 并联至公共 GND)
  PLC DO 输出2 ──► 黄灯 A1
  PLC DO 输出3 ──► 红灯 A1
  PLC DO 输出4 ──► 蜂鸣器 A1
```

### 6.3 急停按钮与按钮开关

![急停按钮](/img/in-post/electrical-components/emergency-stop-button.png)

#### 1. 工作原理
**急停按钮（Emergency Stop Push Button, E-Stop）**是设备安全回路中最重要的人机接口元件。按下后物理自锁，必须旋转或拉出才能复位。**急停回路必须采用常闭（NC）型接线**，构成"安全型设计（Fail-Safe）"——即断线或断电时同样触发急停。

急停按钮接线规范（国际标准 ISO 13850 / IEC 60947-5-5）：
```text
  安全回路：
  +24V ──► 急停 NC触点1 ──► 急停 NC触点2 ──► PLC 安全输入 / 安全继电器
  任意急停按钮压下 → 回路断开 → 控制系统触发紧急停止动作
```

**普通按钮开关**用于手动启动/停止、点动、复位等操作，分为常开（NO）和常闭（NC）型，以及带自锁功能的钥匙开关（防误操作）。

#### 2. 普通按钮接线图
```text
  启动按钮 (NO 型):
  +24V ──► 启动按钮 NO触点 ──► PLC DI（检测到高电平 = 启动信号）

  停止按钮 (NC 型):
  +24V ──► 停止按钮 NC触点 ──► PLC DI（高电平 = 正常；低电平 = 停止）
```

#### 3. 配置建议
- **每台设备**至少配备 1 个外部急停按钮（操作面板正面），大型设备每个操作侧均配置。
- **安全继电器**：在对安全要求高的场合（CE 认证设备、人机协作工站），急停按钮须接入**安全继电器**（如皮尔兹 PILZ PNOZ、施耐德 XPSAF）而非直接接 PLC，满足 ISO 13849 安全等级 PLd 要求。

## 七、现场高效连接与走线规范

在设备高速往复运动中，现场的接线可靠性直接决定了整机的 MTBF（平均无故障工作时间）。

### 7.1 8位集线分线接线盒 — 胜蓝电气 H450 系列

![8位接线盒](/img/in-post/electrical-components/connector-junction-box.png)

#### 1. 为什么必须使用 8 位接线盒？
在以往的布线设计中，如果 Z 轴机械手臂上有 8 个传感器（限位、原点、接近开关等），工程师需要把 8 根 3 芯电缆（共 24 根芯线）一股脑塞进坦克拖链拉回柜内的端子排上。这会带来三大痛点：
1. **拖链塞满**：电缆太多导致拖链粗重，弯曲半径大，轴电机负荷超载。
2. **磨损折断**：普通电缆在拖链长期往复磨损下极易发生隐蔽性折断（铜丝断裂但护套未破），排查极其耗时。
3. **维护地狱**：更换其中一个损坏的传感器需要拆线一整条拖链，调试耗时以天计算。

**胜蓝电气 (SIRON)** 的 H450 8位接线盒解决了这一问题。它安装在机构的机械手板上（活动末端）：
- 8 个传感器的短线（M8 插头）直接插在接线盒的 8 个端口上。
- 接线盒内部将 24V 极性、GND 极性公共端进行了并联。
- 最后仅通过一根**拖链专用高柔性屏蔽电缆**（通常为 10 芯）引出，通过拖链穿回主控模块。

#### 2. 接线盒集线与电气回路走向
```text
  [PM-Y45 传感器 1 (M8插头)] ──► [PORT 1] ──┐
  [PM-Y45 传感器 2 (M8插头)] ──► [PORT 2] ──┼─► [H450内部汇流并线] ── (10芯高柔电缆) ──► [远程IO LHS3-3200]
  ...                                      │   - 1脚并接(24V) ──────► 棕 ────► V+
  [磁开关 8 (M8插头)]          ──► [PORT 8] ──┘   - 3脚并接(GND) ──────► 蓝 ────► V-
                                                 - 各信号引脚 (4脚) ───► 对应彩色信号线 ──► IN1 ~ IN8
```

#### 3. 项目实战配置
- **交错上料组件 (4 PCS)**：型号 **H450-8TF-5000/200**（出厂自带 5 米拖链线，尾部 200mm 分线引脚）。主要用于收纳双直线电机上的 PM-Y45 信号，穿出上料站拖链后，统一插至 `LHS3-3200-XET1`。
- **中转、定位、下料、流线各组件**：大量配备 **H450-8-F** 插座面板盒，方便气动机械手的模块化快速拆装维护。

### 7.2 端子排

![端子排](/img/in-post/electrical-components/terminal-block.png)

#### 1. 工作原理
端子排（Terminal Block）是电控柜内导线连接与分配的标准化基础元件。通过将多根导线固定在同一接线端子上，实现导线间的可靠电气连接，同时便于测量、诊断和维护时的插拔操作。

常见端子排类型：

| 类型 | 典型产品 | 特点 |
|:---|:---|:---|
| **弹簧插拔式端子排** | 菲尼克斯 UTTB / PTSA | 免工具插拔，工程效率高 |
| **螺钉夹紧型端子排** | 魏德米勒 W-Series | 传统可靠，适用大电流 |
| **贯通型端子排** | 正泰 JUT1-4 | 通用性强，成本低 |
| **分断型端子排（保险丝型）** | 菲尼克斯 UTTB 4 SI | 带保险丝，每路独立保护 |
| **导轨安装 DIN 端子排** | 各通用型 | 卡入 35mm 标准 DIN 导轨 |

#### 2. 端子排标识与接线规范
```text
  电控柜端子排标识规范：
  ┌─────────────────────────────────────────────────────────┐
  │  端子编号 (如 X1-01)   导线颜色规范:                    │
  │  X1: 第1组端子排号     红/棕色线: 24V DC 正极           │
  │  01: 第1个端子位号     蓝色线:    0V GND (直流负极)     │
  │                        黄绿双色:  PE 保护地线            │
  │  线号管（线号码管）    黑色线:    AC 火线 L              │
  │  每根线两端套标签，    白色线:    AC 零线 N              │
  │  标注 「去向-线号」     灰色线:    信号线                 │
  └─────────────────────────────────────────────────────────┘
```

#### 3. 工程规范要点
- **线径选择**：24V 控制信号线选 0.5~1mm² 多股铜芯线；电机动力线根据电流选 1.5~6mm²；PE 保护地线截面积须 ≥ 等效相线截面积。
- **导线标识**：每根导线两端套热缩印字线号管（PVC 或尼龙），清晰标注线号（如"PLC-Q0.0-SV1"），确保维护时一眼可识别去向。

### 7.3 拖链与工业线缆

![拖链](/img/in-post/electrical-components/cable-drag-chain.png)

#### 1. 工作原理
拖链（Cable Drag Chain / Energy Chain）是一种将随动运动轴电缆束保护在链节内部、随轴往复运动的电缆导向保护装置。典型品牌为德国 **易格斯（igus）**，其工程塑料链节耐油污、免润滑、寿命超千万次。

拖链选型关键参数：
- **内部截面尺寸（宽×高）**：确保电缆总截面积不超过拖链内截面的 60%，预留散热与弯曲余量。
- **弯曲半径（R）**：通常为拖链链节高度的 2~3 倍，确保电缆在弯曲处不折断。对应选用**拖链专用高柔性电缆（Chainflex）**，普通 PVC 电缆在拖链中往复弯曲 10 万次即断芯。

工业线缆关键类型：
| 类型 | 特点 | 用途 |
|:---|:---|:---|
| 拖链高柔性多芯控制电缆 | 细股镀锡铜芯，超柔绝缘材料 | 拖链内传感器/IO 信号 |
| 屏蔽双绞电缆 | 铝箔 + 铜网双重屏蔽 | 编码器、模拟量信号 |
| 伺服动力电缆 | 高柔性，EMC 屏蔽 | 伺服电机 UVW 相线 |
| 网线（CAT6 SFTP）| 屏蔽对绞，带铝箔屏蔽 | EtherCAT、GigE 相机 |

#### 2. 布线规范
- **强弱电分离**：动力线（电机相线）与信号线（传感器、编码器）须走独立线槽，间距 ≥ 200mm 或中间加隔板，防止电磁感应干扰。
- **拖链内电缆预留**：拖链固定端电缆须留有 1~1.5 倍弯曲半径的余量（自然弧度），避免运动时拉扯断裂。

### 7.4 M12 连接器与航空插头

![M12 连接器](/img/in-post/electrical-components/connector-m12.png)

#### 1. 工作原理
工业连接器是现场快速接线与模块化维护的关键工具。相比于接线端子板的逐芯接线，连接器实现了整体拔插，大大缩短了更换传感器/执行器的维修时间。

**M12 连接器**（螺纹锁定，直径 12mm）是工业传感器与 IO 模块的事实标准：
- **3芯**（A型）：用于 2线制传感器（+V、Signal、GND）。
- **4芯**（A型）：用于标准 NPN/PNP 型 3线制传感器（+V、OUT、GND + 屏蔽/空）。
- **5芯**（B型）：用于 4线制传感器（+V、OUT1、OUT2、GND）或执行器。
- **8芯**：用于 IO 模块本地端口，一根网线替代 8 路传感器布线。

**航空插头（Circular Connector）**用于大电流或多芯高密度场合：
- 伺服电机电源插头（4芯，U/V/W + PE）
- 相机 I/O 触发插头（8~16芯 Hirose HR10 / M12 编码 D型）
- 快插式拖链接线头（方便整体拆卸维护）

#### 2. M12 传感器接线图（A型 4芯）
```text
  M12 母座 (底部视图)
  ┌────────────────────────────────┐
  │   Pin 1 (棕) ──► +24V          │
  │   Pin 2 (白) ──► 信号 OUT2     │
  │   Pin 3 (蓝) ──► 0V GND        │
  │   Pin 4 (黑) ──► 信号 OUT1 ────►  PLC/远程 IO 输入端
  └────────────────────────────────┘
```

#### 3. 应用场景
- **机械手模块化快插**：在机械手法兰板上焊接 M12 母座，传感器通过 M12 公头插头插接，2 分钟内可完整更换整套传感器组件，实现零停机维护。
- **IO 模块本地连接**：灵猴 LHS3 模块的传感器输入端口为 M12 格式，传感器电缆使用 M12 预制线缆，免工具快速插接。
