## 一、传感器类

传感器是上位机获取现场状态的第一道关口，负责将物理量（位置、压力、温度、图像等）转换为 PLC / 运动控制卡可读取的电信号。

### 光电传感器

光电传感器（Photoelectric Sensor）通过发射和接收光束来检测物体的有无、位置或距离，是工业产线上使用最为普遍的传感器之一。常见光源为红外光或可见红光（激光型）。

根据发光与受光的结构，光电传感器主要分为以下三种类型：

| 类型 | 原理 | 特点 |
| :--- | :--- | :--- |
| **对射型** | 发射器与接收器分体，物体遮断光束时触发 | 检测距离最长，可靠性最高 |
| **反射型** | 发射与接收同体，利用反光板返回光束，物体遮断时触发 | 安装方便，只需布线一侧 |
| **槽型（U型）** | 发射与接收集成在U形槽的两臂内 | 检测距离固定，精度高，适合小物体和位置定位 |

### 1.1 松下 EX-14A — 反射型光电传感器

![松下 EX-14A 反射型光电传感器](/img/in-post/electrical-components/ex-14a-retroreflective.png)

#### 工作原理

EX-14A 属于**回归反射型（Retroreflective）**，发射器和接收器集成在同一外壳中。使用时需要在传感器对面安装一个**反射板**（棱镜板），传感器发出的光束打在反射板上后原路返回，接收器持续接收到该返回光。当被检测物体**遮挡**了光束路径时，接收器收到的光强下降，从而触发输出信号。

```text
  [EX-14A 发射+接收]  ──── 光束 ────►  [反射板]
                      ◄─── 反射 ────

  有物体经过时：
  [EX-14A]  ──── 光束 ────► [物体] (光被遮断，输出触发)
```

#### 主要特性

- **检测方式**：回归反射型
- **有效检测距离**：约 0.1m ~ 2m（取决于反射板规格）
- **光源**：红外 LED（不可见光）
- **输出形式**：NPN 集电极开路 / PNP（根据型号后缀区分）
- **响应时间**：约 1ms

#### 典型接线（NPN型，3线式）

```text
  棕色（Brown）  ──── +12~24V DC 电源正
  蓝色（Blue）   ──── 0V 电源负（GND）
  黑色（Black）  ──── 信号输出（低电平有效，接 PLC 输入点）
```

#### 应用场景

- 流水线上检测较大尺寸的工件是否到位
- 检测托盘、载具进出工站
- 适合不能在对面布线，只能单侧安装的场合

### 1.2 松下 EX-13EA — 对射型光电传感器

![松下 EX-13EA 对射型光电传感器](/img/in-post/electrical-components/ex-13ea-through-beam.png)

#### 工作原理

EX-13EA 属于**对射型（Through-Beam）**，发射器（Emitter）和接收器（Receiver）是**两个独立的元件**，分别安装在检测区域的两侧。发射器持续发出光束，接收器持续接收。当被检测物体进入两者之间并**遮断光束**时，接收器输出信号发生变化。

图中左侧为**发射器（EX-11EI）**，右侧为**接收器（EX-11EBD）**，两者配对使用。

```text
  [发射器]  ──── 光束 ────────►  [接收器]   → 无物体：正常接收，输出 OFF
  [发射器]  ──── 光束 ── [物体] X [接收器]  → 有物体：光被遮断，输出 ON
```

#### 主要特性

- **检测方式**：对射型（穿透型）
- **有效检测距离**：可达数米，远超反射型
- **光源**：红外 LED 或红色可见光 LED
- **抗干扰能力**：最强（光路直接，不受环境反射干扰）
- **安装要求**：需要两侧各布线，发射/接收必须精准对准

#### 典型接线

对射型有**发射器和接收器两组接线**：

```text
  发射器（Emitter）：
    棕色 ──── +12~24V
    蓝色 ──── GND

  接收器（Receiver）：
    棕色 ──── +12~24V
    蓝色 ──── GND
    黑色 ──── 信号输出（接 PLC 输入点）
```

#### 应用场景

- 检测透明薄膜、玻璃等反射型难以检测的物料
- 长距离物体有无检测（如仓库货架出入口）
- 高可靠性要求场合，对射型误判率极低
- 检测细小线材、丝线等是否断裂

### 1.3 松下 PM-Y45U — 槽型光电传感器

![松下 PM-Y45U 槽型光电传感器](/img/in-post/electrical-components/pm-y45u-slot.png)

#### 工作原理

PM-Y45U 属于**槽型（Slot Type / U型）**光电传感器。其发射器和接收器集成在一个 **U 形槽**的两侧臂内，检测槽宽度固定（PM-Y45U 槽宽约 **5mm**）。被检测物体穿过槽口时，光束被遮断，从而触发输出。

```text
  ┌─────────────────┐
  │  发射  ║   ║ 接收 │
  │  LED  ║ 槽 ║ PD  │
  └─────────────────┘
       ↑          ↑
       │  ← 5mm → │
       └──光束穿越──┘
```

#### 主要特性

- **检测方式**：U型对射（发射接收一体）
- **槽宽**：5mm（固定，物理限制）
- **检测对象**：穿越槽口的细小物体（叶片、挡片、编码盘齿）
- **重复定位精度**：极高，可做位置基准
- **输出形式**：NPN / PNP 集电极开路

#### 典型接线（3线式）

```text
  棕色 ──── +5~24V DC
  蓝色 ──── GND
  黑色 ──── 信号输出（接 PLC 输入点）
```

#### 应用场景

- **原点检测 / 零位复位**：机械轴上固定一个挡片，运动到槽型传感器时触发，作为回零信号
- **编码计数**：配合编码盘的齿孔，实现转速或位移测量
- **限位检测**：作为轴的行程极限位置开关
- **小零件到位检测**：检测小尺寸工件（如螺丝、销钉）是否进入固定位置

### 光电传感器选型对比总结

| 对比项 | EX-14A 反射型 | EX-13EA 对射型 | PM-Y45U 槽型 |
| :--- | :---: | :---: | :---: |
| **接线侧数** | 单侧 | 双侧（发射+接收） | 单侧 |
| **是否需要反光板** | ✅ 需要 | ❌ 不需要 | ❌ 不需要 |
| **检测距离** | 中等（~2m） | 最长（数米） | 极短（固定槽宽） |
| **抗干扰能力** | 一般 | 最强 | 强（封闭槽内） |
| **对透明物体检测** | 较难 | 容易 | 容易 |
| **精度/可重复性** | 一般 | 一般 | 极高 |
| **典型用途** | 工件到位、载具检测 | 高可靠检测、透明物料 | 原点/限位/计数 |

---

> **采购提示**：松下 EX 系列传感器型号后缀含义：
> - 无后缀或 `-P`：PNP 输出（输出高电平，适合西门子等欧系 PLC）
> - `-N` 或 `-A`：NPN 输出（输出低电平，适合三菱、欧姆龙等日系 PLC）
> - 确认供电电压范围（通常 12~24V DC），以及是否有指示灯（方便现场调试）

### 1.4 近接传感器（接近传感器）

![欧姆龙 E2B 近接传感器](/img/in-post/electrical-components/sensor-proximity.png)

#### 工作原理

近接传感器（Inductive Proximity Sensor）利用**电磁感应**原理工作。传感器内部振荡电路产生交变磁场，当**金属物体**进入检测范围时，涡流效应使振荡幅度衰减，触发输出信号翻转。

```text
  [近接传感器]  ──── 交变磁场 ────►  金属工件靠近
  无金属：振荡正常 → 输出 OFF
  有金属：振荡衰减 → 输出 ON
```

仅对金属有效，对塑料、木材、液体等非金属物体无反应（需选用电容式或其他类型）。

#### 主要特性

- **检测对象**：铁、钢、铝、铜等金属（屏蔽型仅检测正面金属）
- **典型型号**：欧姆龙 E2B 系列、西克 IME 系列
- **检测距离**：M8 约 1.5mm，M12 约 4mm，M18 约 8mm（与直径相关）
- **输出形式**：NPN / PNP 集电极开路，常开 NO / 常闭 NC
- **安装方式**：螺纹埋入式（需按标称距离留余量，避免撞坏）

#### 典型接线（3线式 NPN）

```text
  棕色（Brown）  ──── +12~24V DC
  蓝色（Blue）   ──── 0V（GND）
  黑色（Black）  ──── 信号输出（低电平有效，接 PLC 输入点）
```

#### 应用场景

- 检测金属治具、夹具是否到位
- 旋转轴上检测金属挡片位置（替代机械限位开关）
- 输送线上检测金属工件有无
- 气缸活塞杆末端金属检测（需确认安装位置）

### 1.5 磁性传感器（气缸磁开关）

![SMC D-M9N 磁性传感器](/img/in-post/electrical-components/sensor-magnetic.png)

#### 工作原理

磁性传感器（Reed Switch / 磁簧开关）利用**永磁体**磁场触发干簧管闭合。气缸活塞上安装永磁环，当活塞运动到传感器安装位置时，磁场使干簧管内触点吸合，输出信号。

```text
  气缸缩回：磁环远离传感器 → 输出 OFF
  气缸伸出：磁环经过传感器 → 输出 ON（确认到位）
```

#### 主要特性

- **典型型号**：SMC D-M9N / D-M9B、费斯托 SME 系列
- **安装方式**：卡扣式固定在气缸缸体沟槽上，可沿轴向滑动调节
- **输出形式**：2线式（与 PLC 输入串联）或 3线式 NPN/PNP
- **检测对象**：仅对气缸内置永磁环有效
- **优点**：非接触、寿命长、不受油污灰尘影响

#### 典型接线（3线式）

```text
  棕色 ──── +12~24V
  蓝色 ──── GND
  黑色 ──── 信号输出（接 PLC 输入点）
```

2线式直接串联在 PLC 输入回路中，需注意 PLC 输入类型与电流匹配。

#### 应用场景

- 气缸**伸出到位 / 缩回到位**确认（双磁开关分别检测两端）
- 替代机械式行程开关，避免撞击磨损
- 多工位夹紧气缸的互锁信号
- 与电磁阀联锁，防止气缸未到位就执行下一动作

### 1.6 压力传感器

![松下 DP-100 压力传感器](/img/in-post/electrical-components/sensor-pressure.png)

#### 工作原理

数字压力传感器将气压或液压转换为电信号。内部传感元件（应变片、陶瓷或 MEMS）感受压力变化，经放大和 A/D 转换后输出开关量或模拟量。

```text
  气源 ──► [压力传感器] ──► 开关输出 / 模拟量 4~20mA / 0~10V
              │
              └── 设定值比较 → 高压报警 / 低压报警
```

#### 主要特性

- **典型型号**：松下 DP-100 系列、SMC ZSE30 / ISE30 系列
- **量程**：负压 -100kPa ~ 正压 1MPa（按型号区分）
- **输出形式**：NPN/PNP 开关量 + 模拟量（部分型号）
- **显示**：自带 LED 数显，可现场设定阈值
- **接口**：M5 / Rc1/8 等气动螺纹接口

#### 典型接线

```text
  电源：棕色 +24V / 蓝色 GND
  开关输出：黑色（低压报警）/ 白色（高压报警，视型号）
  模拟量：4~20mA 两线制接入 AI 模块
```

#### 应用场景

- 真空吸盘负压检测（确认吸附成功）
- 主气路压力监控（低于设定值报警停机）
- 夹具液压回路压力确认
- 涂胶 / 点胶设备气压稳定监控

### 1.7 温度传感器

![PT100 热电阻温度传感器](/img/in-post/electrical-components/sensor-temperature.png)

#### 工作原理

**PT100 热电阻**：铂电阻阻值随温度线性变化（0°C 时 100Ω），通过测量电阻推算温度。

**K 型热电偶**：两种不同金属导体接合处产生热电势（Seebeck 效应），温差越大电势越高。

```text
  PT100：  温度变化 → 电阻变化 → 变送器 → 4~20mA / Modbus
  热电偶： 温差产生 mV 信号 → 温控表 / AI 模块采集
```

#### 主要特性

| 类型 | 测温范围 | 精度 | 接线 |
| :--- | :--- | :--- | :--- |
| PT100（3线制） | -200~600°C | 高（Class A ±0.15°C） | 3线补偿引线电阻 |
| K 型热电偶 | 0~1200°C | 中（±1.5°C 或 ±0.4%） | 正负极不可接反 |

#### 典型接线

```text
  PT100 三线制：
    红线 ──── 电源正（变送器）
    白线 ──── 信号回路
    蓝线 ──── 补偿线（接传感端）

  热电偶：
    正极（红/黄）─── 温控表 +
    负极（白/蓝）─── 温控表 -
```

#### 应用场景

- 烘箱、热压、焊接工位温度监控
- 电机 / 伺服驱动器过热保护
- 液冷设备进出口水温检测
- 配合温控表实现 PID 恒温控制

### 1.8 视觉传感器 / 工业相机

![海康威视工业相机](/img/in-post/electrical-components/sensor-vision-camera.png)

#### 工作原理

工业相机将光学图像转换为数字信号，配合视觉软件完成**定位、测量、识别、缺陷检测**。上位机通过 GigE / USB3.0 接收图像，运行算法后输出坐标、角度、OK/NG 结果给 PLC。

```text
  [相机] ── 图像 ──► [工控机/视觉控制器] ── 算法处理 ──► 结果给 PLC
                                              │
                                              ├── X/Y/θ 坐标
                                              └── OK / NG 信号
```

#### 主要特性

- **典型品牌**：海康威视 MV-CA 系列、大恒 MER 系列
- **接口**：GigE 千兆网（最常用）、USB3.0、Camera Link
- **分辨率**：130万 ~ 2000万像素（按检测精度选型）
- **镜头**：C 口定焦 / 远心镜头（高精度测量用）
- **光源**：环形光、条形光、同轴光（与相机配套使用）

#### 典型接线

```text
  相机供电：12V / 24V（PoE 相机通过网线供电）
  通信：   网线接工控机网口（GigE），配置固定 IP
  触发：   PLC 输出 → 相机 IO 触发拍照
  结果：   视觉软件 TCP/Modbus/IO 输出给 PLC
```

#### 应用场景

- 零件定位引导机器人抓取
- 标签 / 字符 OCR 识别
- 表面划痕、缺件、异物缺陷检测
- 尺寸测量与公差判定

### 1.9 光纤传感器

![松下 FX-500 光纤传感器](/img/in-post/electrical-components/sensor-fiber.png)

#### 工作原理

光纤传感器由**放大器单元**和**光纤探头**两部分组成。放大器发射 LED 光，经光纤传导至探头前端，接收反射光或透射光强度变化来判断物体状态。光纤本体无电路，可弯曲进入狭小空间。

```text
  [放大器] ──光纤──► [探头前端] ── 检测区域
  反射型：物体反射光回到接收光纤
  对射型：发射/接收光纤相对，物体遮断光路
```

#### 主要特性

- **典型型号**：松下 FX-500 / FX-551 系列
- **探头类型**：同轴反射、区域对射、限定反射（多种头可更换）
- **检测距离**：数 mm ~ 数百 mm（取决于光纤头型号）
- **优势**：探头极小、耐弯曲、适合狭窄空间
- **输出**：NPN / PNP，部分型号支持模拟量

#### 典型接线（3线式）

```text
  棕色 ──── +12~24V
  蓝色 ──── GND
  黑色 ──── 信号输出
```

#### 应用场景

- 半导体 / 精密零件微小物体检测
- 狭窄缝隙内的液位检测
- 高温环境（探头耐高温，放大器远离热源）
- 振动盘出料计数

---

### 1.10 色标传感器

![基恩士 CZ 系列色标传感器](/img/in-post/electrical-components/sensor-color-mark.png)

#### 工作原理

色标传感器发射红 / 绿 / 蓝等特定波长 LED 光，接收物体表面反射光，通过比较**颜色或灰度差异**识别色标带、印刷标记或材料色差。

```text
  [色标传感器] ── LED ──► 物体表面
                    ◄── 反射光强度/颜色比较
  识别到标记色 → 输出 ON（用于定位、计数、分选）
```

#### 主要特性

- **典型型号**：基恩士 CZ-V 系列、欧姆龙 E3S-DC 色标型
- **检测方式**：色标跟踪、色差比较、灰度阈值
- **响应速度**：最快可达 50μs 级（高速产线）
- ** teach 功能**：按键示教目标颜色，自动设定阈值

#### 典型接线（3线式）

```text
  棕色 ──── +12~24V
  蓝色 ──── GND
  黑色 ──── 信号输出
```

#### 应用场景

- 包装膜色标定位（制袋、横封切割）
- 标签接头检测
- 不同颜色工件分选
- 印刷品套色偏差检测

---

### 1.11 编码器

![欧姆龙 E6B2 增量型编码器](/img/in-post/electrical-components/sensor-encoder.png)

#### 工作原理

增量型编码器随轴旋转输出 A/B 两路**相位差 90°** 的方波脉冲，通过脉冲计数计算位移或转速。Z 相（零位）每转一圈输出一个脉冲，用于原点定位。

```text
  旋转 ──► A 相：▁▃▁▃▁▃
           B 相： ▁▁▃▁▁▃  （A 超前 B = 正转）
           Z 相： ▁───────▁ （每圈一个）
  脉冲数 × 分辨率 = 位移量
```

#### 主要特性

- **典型型号**：欧姆龙 E6B2-C 系列（增量型）
- **分辨率**：100 ~ 3600 P/R（每转脉冲数）
- **输出**：集电极开路（需外接上拉）或 线驱动（RS-422，抗干扰强）
- **轴径**：φ6 / φ8 / φ10 多种规格
- **供电**：DC 5V / 12~24V

#### 典型接线

```text
  棕色 ──── +12~24V
  蓝色 ──── GND
  黑色 ──── A 相
  白色 ──── B 相
  橙色 ──── Z 相（原点）
```

接入 PLC 高速计数器（HSC）或运动控制卡编码器输入端。

#### 应用场景

- 输送带线速度测量
- 卷料张力控制（收放卷直径计算）
- 手动轴手轮位置反馈
- 配合伺服实现全闭环（外接编码器校验）

## 二、执行元件类

执行元件负责将控制信号转化为机械运动或流体动作，是设备"动手"的部分。

### 2.1 电磁阀

![SMC SY 系列电磁阀](/img/in-post/electrical-components/actuator-solenoid-valve.png)

#### 工作原理

电磁阀通过线圈通电产生磁力，推动阀芯换向，控制压缩空气的流向，从而驱动气缸伸出或缩回。5 通 2 位阀是最常用的气缸控制阀。

```text
  线圈 OFF：气源 → P→A，B→排气（气缸缩回）
  线圈 ON ：气源 → P→B，A→排气（气缸伸出）
```

#### 主要特性

- **典型型号**：SMC SY3000 / SY5000、费斯托 MFH 系列
- **通径**：SY3（小流量）/ SY5（标准）/ SY7（大流量）
- **线圈电压**：DC 24V（工业最常用）、AC 110V / 220V
- **接线**：DIN 插座式 / 直接引线，PLC 输出经中间继电器驱动
- **功能**：5/2 单电控、5/2 双电控、3/2 单电控

#### 典型接线

```text
  线圈两端 ──── +24V / 0V（经 PLC 晶体管输出或继电器触点）
  气管：P 接气源，A/B 接气缸两腔，R/S 排气
```

#### 应用场景

- 夹紧气缸、顶升气缸、阻挡气缸控制
- 真空发生器、吹气清料
- 气动手指（平行夹爪）驱动
- 与磁性传感器联锁确认气缸到位

---

### 2.2 气缸

![SMC CQ2 气缸](/img/in-post/electrical-components/actuator-cylinder.png)

#### 工作原理

气缸将压缩空气的压力能转化为直线往复机械运动。单作用气缸一端弹簧复位；双作用气缸靠换向阀控制两端进气实现双向运动。

```text
  双作用气缸：
  进气 A 腔 → 活塞杆伸出
  进气 B 腔 → 活塞杆缩回
```

#### 主要特性

- **典型型号**：SMC CQ2（薄型）、CDQ2、标准型 CG1
- **缸径**：φ6 ~ φ100（推力与缸径平方成正比）
- **行程**：10 ~ 500mm（按结构选型）
- **安装**：基本型、脚座、法兰、耳轴、耳环
- **缓冲**：橡胶缓冲 / 气缓冲（高速运动必配）

#### 典型接线（气路）

```text
  气源（0.5MPa）── 过滤减压阀 ── 电磁阀 ── A/B 腔
  磁性传感器卡在缸体沟槽检测到位
```

#### 应用场景

- 工件夹紧、顶升、推送
- 阻挡分料、升降平台
- 压装、铆接（需计算推力 = 压力 × 面积）
- 配合真空吸盘上下运动

---

### 2.3 伺服电机 + 驱动器

![松下 MINAS A6 伺服系统](/img/in-post/electrical-components/actuator-servo-motor.png)

#### 工作原理

伺服系统由**伺服电机 + 伺服驱动器**组成。驱动器接收脉冲 / 模拟量 / 总线位置指令，通过闭环 PID 控制电机转速和力矩，实现精确定位。编码器反馈实际位置，误差实时修正。

```text
  PLC / 运动控制卡 ── 脉冲/总线 ──► [驱动器] ──► [伺服电机]
                                      ▲              │
                                      └── 编码器反馈 ─┘
```

#### 主要特性

- **典型型号**：松下 MINAS A6（A6B 脉冲型 / A6N EtherCAT）、三菱 MR-J4
- **功率**：50W ~ 15kW
- **控制模式**：位置 / 速度 / 力矩
- **通信**：脉冲+方向、模拟量、EtherCAT、CC-Link
- **精度**：定位精度可达 ±0.01mm 级（取决于机械结构）

#### 典型接线

```text
  动力线：U/V/W 接驱动器输出
  编码器线：专用电缆接驱动器 CN2
  控制线：脉冲 PUL/SIGN/SON 接 PLC 高速输出或运动控制卡
  电源：单相/三相 AC 220V 接驱动器主回路
```

#### 应用场景

- XYZ 龙门定位、多轴插补运动
- 高速贴标、飞达送料
- 精密丝杠平台驱动
- 替代步进电机的高精度场合

---

### 2.4 步进电机 + 驱动器

![雷赛 86 步进电机与 DM542 驱动器](/img/in-post/electrical-components/actuator-stepper-motor.png)

#### 工作原理

步进电机按脉冲数转动固定角度（步距角），无编码器反馈，属于开环控制。驱动器将脉冲信号转换为相电流，驱动电机绕组按序通电旋转。

```text
  1 个脉冲 → 转动 1 步（如 1.8°/步，200 步/圈）
  细分设置：2000 脉冲/圈 → 精度提高 10 倍
```

#### 主要特性

- **典型型号**：雷赛 86CM85、驱动器 DM542 / DM556
- **机座号**：42 / 57 / 86（数字为机身边长 mm）
- **保持力矩**：0.3 ~ 12 N·m
- **成本**：远低于同功率伺服
- **缺点**：高速易失步、发热较大

#### 典型接线

```text
  驱动器 PUL+ / PUL- ──── 脉冲信号（接运动控制卡或 PLC）
  驱动器 DIR+ / DIR- ──── 方向信号
  驱动器 ENA+ / ENA- ──── 使能（可选）
  电机 A+/A-/B+/B- ──── 四根相线
  电源：DC 24~48V 接驱动器（按电机额定电压）
```

#### 应用场景

- 低成本定位轴（送料、分度盘）
- 三轴平台简易运动
- 对速度和精度要求不高的场合
- 教学演示与原型验证

---

### 2.5 变频器

![汇川 MD310 变频器](/img/in-post/electrical-components/actuator-inverter.png)

#### 工作原理

变频器将工频交流电（50Hz）整流为直流，再逆变为频率可调的三相交流电，从而改变电机转速。通过 V/f 控制或矢量控制实现速度调节。

```text
  AC 380V ──► [整流] ── DC 母线 ── [逆变] ──► 变频三相电 ──► 异步电机
  设定频率 50Hz → 额定转速；25Hz → 半速
```

#### 主要特性

- **典型型号**：汇川 MD310、台达 VFD-M / VFD-E 系列
- **功率**：0.4kW ~ 数百 kW
- **控制**：端子启停、模拟量调速、Modbus RS485 通信
- **保护**：过流、过压、过热、缺相
- **制动**：需外接制动电阻（快速减速场合）

#### 典型接线

```text
  R/S/T ──── 三相 AC 380V 输入
  U/V/W ──── 接三相异步电机
  +24V / COM / DI1~DI4 ──── 启停、正反转（接 PLC 继电器）
  AI / AO ──── 模拟量速度给定 / 反馈
  RS485 A/B ──── Modbus 通信
```

#### 应用场景

- 输送带无级调速
- 风机、水泵节能控制
- 搅拌、涂布辊速同步
- 老设备电机改造（替代机械变速）

## 三、控制元件类

控制元件是电气系统的"大脑"和"神经中枢"，负责逻辑运算、信号放大与电力分配。

### 3.1 PLC

![西门子 S7-1200 PLC](/img/in-post/electrical-components/control-plc.png)

#### 工作原理

PLC（可编程逻辑控制器）循环执行用户程序：读取输入状态 → 逻辑运算 → 更新输出状态。通过梯形图 / ST 语言定义设备动作逻辑，替代传统继电器硬接线。

```text
  扫描周期：
  读 DI → 执行程序（梯形图） → 写 DO → 通信处理 → 重复
```

#### 主要特性

- **典型型号**：西门子 S7-1200（1214C 最常用）、三菱 FX5U
- **I/O 点数**：14 ~ 数百点（可扩展模块）
- **通信**：PROFINET、EtherNet/IP、Modbus TCP、RS485
- **编程**：TIA Portal（西门子）、GX Works3（三菱）
- **上位机连接**：通过以太网与 C# / WinForm 上位机交换数据

#### 典型接线

```text
  L+/M ──── DC 24V 电源
  DI 端子 ──── 传感器 NPN 输出（共源）或 PNP（共漏）
  DO 端子 ──── 继电器线圈 / 电磁阀（经中间继电器）
  以太网口 ──── 网线接交换机 / 上位机
```

#### 应用场景

- 产线顺序控制、互锁逻辑
- 气缸 / 阀门动作时序
- 与 HMI、上位机、视觉系统通信
- 替代继电器实现复杂逻辑

---

### 3.2 运动控制卡

![雷赛 MC8040 运动控制卡](/img/in-post/electrical-components/control-motion-card.png)

#### 工作原理

运动控制卡插入工控机 PCI / PCIe 槽，由上位机软件直接发送脉冲或总线指令，控制多轴伺服 / 步进电机运动。支持直线插补、圆弧插补、电子齿轮、高速锁存等高级功能。

```text
  C# 上位机 ── API 调用 ──► [运动控制卡] ── 脉冲/总线 ──► 伺服驱动器
```

#### 主要特性

- **典型型号**：雷赛 MC8040（4轴脉冲）、固高 GT-400
- **轴数**：2 ~ 64 轴（按卡型号）
- **脉冲频率**：最高 4MHz 级
- **功能**：插补、回零、限位、急停、PDO 总线控制
- **接口**：脉冲+方向、编码器反馈、通用 IO

#### 典型接线

```text
  控制卡 ── 脉冲线 ──► 各轴伺服驱动器
  控制卡 ── 编码器线 ◄── 外部编码器（可选）
  控制卡 ── IO 线 ──► 限位开关、原点、急停
  工控机 PCI 槽安装驱动程序
```

#### 应用场景

- 多轴龙门、点胶机、贴片机
- 需要复杂轨迹插补的设备
- 上位机直接控制运动（不经过 PLC）
- 视觉引导抓取坐标系运动

---

### 3.3 继电器（电磁继电器）

![欧姆龙 MY2N 电磁继电器](/img/in-post/electrical-components/control-relay.png)

#### 工作原理

电磁继电器利用线圈通电产生磁场，吸合衔铁带动触点闭合/断开，实现**小电流控制大电流**的电气隔离。PLC 输出电流通常仅数百 mA，无法直接驱动电磁阀或大功率负载，需经继电器中转。

```text
  PLC 输出（小电流）── 线圈 ── 触点 ── 电磁阀 / 接触器（大电流）
```

#### 主要特性

- **典型型号**：欧姆龙 MY2N-GS（8脚）、松下 ALZ11
- **触点容量**：5A ~ 10A（AC 250V / DC 30V）
- **线圈电压**：DC 24V（与 PLC 匹配）
- **底座**：PYF08A 等插座底座，方便更换
- **寿命**：机械寿命千万次级

#### 典型接线

```text
  线圈两端 ──── PLC DO 输出 +24V / COM
  常开触点 ──── 负载回路（电磁阀线圈、指示灯等）
```

#### 应用场景

- PLC 输出驱动电磁阀线圈
- 信号隔离与放大
- 多路分配（一对多触点）
- 交流负载与直流控制回路隔离

---

### 3.4 固态继电器（SSR）

![固态继电器 SSR](/img/in-post/electrical-components/control-ssr.png)

#### 工作原理

SSR 用半导体（可控硅 / 晶体管）代替机械触点，实现无触点开关。输入侧光耦隔离，输出侧控制交流或直流负载通断，无机械磨损、无电弧、动作速度快。

```text
  DC 输入（3~32V）── 光耦 ── 可控硅导通 ── AC 负载通电
```

#### 主要特性

- **典型品牌**：奥托尼克斯、OPTO22、欧姆龙 G3NA
- **负载类型**：交流 SSR（AC 24~480V）、直流 SSR
- **额定电流**：10A ~ 40A（需配散热器）
- **控制电压**：DC 3~32V
- **优点**：静音、高频动作、寿命长

#### 典型接线

```text
  输入 + / - ──── PLC 输出 +24V / COM
  输出端 ──── 串联 AC 负载（加热器、灯泡等）
  注意：交流 SSR 需接散热片，负载电流不超过额定 70%
```

#### 应用场景

- 加热器温控（PWM 调功）
- 频繁动作的交流负载
- 无火花场合（防爆区域）
- 电磁阀驱动（部分直流 SSR 型号）

---

### 3.5 接触器

![施耐德 LC1D 接触器](/img/in-post/electrical-components/control-contactor.png)

#### 工作原理

接触器本质是大容量电磁继电器，用于频繁接通和分断**主电路**（电机、电加热等大功率负载）。主触点承载大电流，辅助触点用于自锁和信号反馈。

```text
  启动按钮 → 线圈通电 → 主触点闭合 → 电机运转
  辅助触点自锁 → 松开按钮后保持运行
  停止按钮 → 线圈断电 → 主触点断开
```

#### 主要特性

- **典型型号**：施耐德 LC1D09~D95、ABB A 系列
- **主触点容量**：9A ~ 95A（AC-3 电机负载）
- **线圈电压**：AC 220V / DC 24V
- **辅助触点**：1NO+1NC 或更多（可扩展）
- **配合**：热继电器实现电机过载保护

#### 典型接线

```text
  主回路：L1/L2/L3 ── 主触点 ── U/V/W 接电机
  控制回路：线圈 A1/A2 接启动/停止按钮与 PLC
  辅助触点：反馈运行状态给 PLC
```

#### 应用场景

- 三相异步电机直接启动 / 星三角启动
- 大功率电加热器主回路控制
- 水泵、风机、搅拌电机
- 配电柜主开关下级控制

---

### 3.6 断路器 / 空气开关

![施耐德 iC65N 断路器](/img/in-post/electrical-components/control-circuit-breaker.png)

#### 工作原理

断路器（MCB）在电路发生过载或短路时，双金属片发热弯曲或电磁脱扣器动作，自动跳闸切断电源，保护线路和设备安全。手动操作可合闸复位。

```text
  正常：手动合闸，电流通过
  过载：双金属片弯曲脱扣 → 跳闸
  短路：电磁瞬时脱扣 → 跳闸
```

#### 主要特性

- **典型型号**：施耐德 iC65N、ABB S200 系列
- **极数**：1P / 2P / 3P / 4P
- **额定电流**：1A ~ 63A（按回路负载选型）
- **曲线**：C 型（通用）、D 型（电机启动）
- **分断能力**：6kA ~ 10kA

#### 典型接线

```text
  进线端 ──── 配电箱总线 / 变压器输出
  出线端 ──── 各回路负载（开关电源、控制回路等）
  按负载电流 × 1.25 倍选型额定电流
```

#### 应用场景

- 电控柜总电源保护
- 各分回路独立保护（24V 电源、220V 控制回路）
- 设备主电源进线
- 配合漏电保护器（漏保）使用

---

### 3.7 远程 IO 模块

![西门子 ET200SP 远程 IO 模块](/img/in-post/electrical-components/control-remote-io.png)

#### 工作原理

远程 IO 模块（Remote I/O / 分布式 IO）将数字量、模拟量采集点延伸到远离主 PLC 的现场，通过**工业总线**与主站通信。主 PLC 或上位机通过总线读写远程模块的 IO 状态，就像访问本地 IO 一样。

```text
  [主 PLC / 上位机] ── 总线（PROFINET / EtherCAT / CC-Link）──► [远程 IO 站]
                                                              ├── DI 模块（传感器）
                                                              ├── DO 模块（电磁阀）
                                                              └── AI/AO 模块（模拟量）
```

#### 主要特性

- **典型型号**：西门子 ET200SP / ET200MP、三菱 CC-Link IE Field、欧姆龙 NX-ECC、菲尼克斯 Axioline F
- **通信协议**：PROFINET、EtherCAT、EtherNet/IP、Modbus TCP、CC-Link
- **模块类型**：数字量输入（DI）、数字量输出（DO）、模拟量输入（AI）、模拟量输出（AO）、高速计数
- **站号**：一条总线上可挂多个远程站，按 IP 或站号寻址
- **优势**：大幅减少长距离拉线到主控柜的电缆，缩短调试周期

#### 典型接线

```text
  通信：  网线 / 专用总线电缆接主 PLC 交换机或总线接口
  电源：  独立 24V 供电（每个远程站需配电）
  DI 端子：传感器 NPN/PNP 接入对应通道
  DO 端子：经继电器或直接驱动电磁阀（视输出类型）
  屏蔽：  通信电缆与动力电缆分开走线，屏蔽层单端接地
```

#### 应用场景

- 大型设备分区布线（主机架在柜内，远程站在设备另一端）
- 多台工位共用一条总线，各工位独立 IO 站
- 输送线、龙门架等跨度大的场合
- 上位机通过 PLC 间接读写远程 IO，实现跨工位状态监控

## 四、人机交互 / 指示元件类

### 4.1 工业触摸屏（HMI）

![威纶通 MT8071iE 触摸屏](/img/in-post/electrical-components/hmi-touchscreen.png)

#### 工作原理

HMI（Human Machine Interface）通过触摸屏显示设备状态、参数设定、报警信息，并将操作指令发送给 PLC。HMI 本身不做逻辑运算，仅作为人机交互界面。

```text
  操作员 ── 触摸 HMI ── 通信 ──► PLC（读写寄存器）
  PLC 状态 ── 通信 ──► HMI 画面显示
```

#### 主要特性

- **典型型号**：威纶通 MT8071iE（7寸）、昆仑通态 TPC7072Gi
- **屏幕尺寸**：4.3 ~ 15 寸
- **通信**：RS485（Modbus RTU）、以太网（Modbus TCP / 西门子 S7）
- **组态软件**：EasyBuilder Pro（威纶通）、MCGS
- **功能**：趋势图、配方、多语言、U 盘升级

#### 典型接线

```text
  24V 电源 ──── HMI 电源端子
  RS485 A/B ──── PLC RS485 口（或网线接以太网口）
  USB ──── 程序下载 / U 盘
```

#### 应用场景

- 设备参数设定与修改
- 运行状态、产量、报警实时显示
- 手动 / 自动模式切换
- 替代大量物理按钮和指示灯

---

### 4.2 指示灯（信号灯）

![施耐德 XB4 指示灯](/img/in-post/electrical-components/indicator-light.png)

#### 工作原理

指示灯内部 LED 发光，通过不同颜色传达设备状态信息。通常由 PLC 输出或继电器触点驱动，向操作员直观显示运行、停止、故障等状态。

```text
  PLC DO = ON → 绿灯亮（运行中）
  PLC DO = ON → 红灯亮（故障报警）
```

#### 主要特性

- **典型型号**：施耐德 XB4 系列、积茂 LT 系列
- **颜色规范**：绿=运行、红=故障/急停、黄=警告、蓝=辅助
- **电压**：AC/DC 24V（与 PLC 输出匹配）
- **安装**：φ22mm 面板开孔（最常见）

#### 典型接线

```text
  指示灯一端 ──── +24V
  指示灯另一端 ──── PLC DO 输出（低侧接法）或继电器触点
```

#### 应用场景

- 电控柜面板状态指示
- 安全门、光栅状态显示
- 与蜂鸣器联动报警
- 手动操作台工位指示

---

### 4.3 蜂鸣器 / 报警灯（三色灯塔）

![三色灯塔报警灯](/img/in-post/electrical-components/buzzer-stack-light.png)

#### 工作原理

三色灯塔由红 / 黄 / 绿（或更多色）独立 LED 层叠组成，配合蜂鸣器发出声光报警。各层独立接线，PLC 按优先级控制不同颜色点亮和蜂鸣。

```text
  绿色层 ON  → 设备正常运行
  黄色层 ON  → 警告（缺料、待确认）
  红色层 ON + 蜂鸣 → 故障停机
```

#### 主要特性

- **层数**：2层 / 3层 / 5层
- **蜂鸣**：间歇蜂鸣 / 连续蜂鸣（可调）
- **电压**：DC 24V
- **安装**：底座固定或壁挂

#### 典型接线

```text
  每层 LED：独立 +24V / PLC DO 控制
  蜂鸣器：  独立 DO 控制（或与红色层联动）
```

#### 应用场景

- 产线顶部状态一目了然
- 安灯系统（Andon）呼叫
- 设备故障声光报警
- 无人值守设备远程状态辨识

### 4.4 急停按钮 / 按钮开关

![施耐德急停按钮](/img/in-post/electrical-components/emergency-stop-button.png)

#### 工作原理

急停按钮（E-Stop）采用**常闭（NC）触点**，正常时触点闭合维持安全回路。按下后蘑菇头锁定，触点断开，切断安全继电器输出，所有运动立即停止。需旋转复位。

```text
  正常：NC 触点闭合 → 安全回路导通 → 设备可运行
  按下：NC 触点断开 → 安全继电器失电 → 全部停止
```

#### 主要特性

- **典型型号**：施耐德 XB5AS542、ABB CE4T-10
- **颜色**：红色蘑菇头 + 黄色底座（强制标准）
- **触点**：1NC / 2NC（安全回路至少 2NC）
- **标准**：符合 ISO 13850 / IEC 60947-5-5
- **安装**：面板开孔 φ22mm，高度便于紧急触及

#### 典型接线

```text
  2 个 NC 触点串联 ──── 安全继电器输入回路（S11/S12）
  不可直接接 PLC 普通 DI 代替安全回路
```

#### 应用场景

- 每台设备至少 1 个急停（国标要求）
- 安全防护门联动
- 与安全继电器 / 安全 PLC 组成安全回路
- 手动操作台、巡检通道

## 五、电源与配电类

### 5.1 开关电源

![明纬 SDR-120-24 开关电源](/img/in-post/electrical-components/power-supply.png)

#### 工作原理

开关电源将 AC 220V 整流后，通过高频开关管斩波、变压器隔离、滤波稳压，输出稳定的 DC 24V。相比线性电源，体积小、效率高（>85%）。

```text
  AC 220V ──► [整流滤波] ── [高频变换] ── [稳压] ──► DC 24V
```

#### 主要特性

- **典型型号**：明纬 SDR-120-24（120W）、NDR-240-24（240W）
- **输出**：DC 24V（工业标准）、5V / 12V / 48V
- **功率**：35W ~ 960W
- **安装**：DIN 导轨安装
- **保护**：过压、过流、短路、过热

#### 典型接线

```text
  L/N ──── AC 220V 输入（经断路器保护）
  +V / -V ──── DC 24V 输出（接端子排分配）
  按总负载电流 × 1.3 倍选型功率
```

#### 应用场景

- 电控柜 DC 24V 总电源
- PLC、传感器、电磁阀、HMI 供电
- 工控机辅助电源
- 每套设备通常 1~2 台（控制回路与动力回路分开）

---

### 5.2 24V 电源分配模块

![菲尼克斯 24V 分配模块](/img/in-post/electrical-components/power-distribution-module.png)

#### 工作原理

电源分配模块将开关电源的 24V 输出分配为多路独立端子，每路带 LED 指示和熔断器保护。某一路短路时仅该路熔断器跳闸，不影响其他回路。

```text
  开关电源 +24V ──► [分配模块] ──┬── 路1：PLC
                                 ├── 路2：传感器
                                 ├── 路3：电磁阀
                                 └── 路4：HMI
```

#### 主要特性

- **典型型号**：菲尼克斯 MINI MCR-SL-UI-UI、魏德米勒 PRO MAX
- **路数**：4 / 8 / 16 路
- **每路电流**：5A ~ 10A
- **指示**：每路 LED 亮/灭状态
- **保护**：可插拔熔断器

#### 典型接线

```text
  进线端 ──── 开关电源 +24V / 0V
  各出线端 ──── 对应回路负载
  0V 汇流排共用
```

#### 应用场景

- 电控柜内规范化配电
- 分回路保护（传感器与阀门分开）
- 故障快速定位（LED 熄灭即该路异常）
- 替代手工并联接线

## 六、连接与布线类

### 6.1 端子排

![菲尼克斯 UTTB 端子排](/img/in-post/electrical-components/terminal-block.png)

> 与 **6.4 IO 端子模块** 的区别：本节端子排为通用配电/转接；IO 端子模块专为 PLC IO 回路设计，带通道 LED 和预制连接器。

#### 工作原理

端子排安装在 DIN 导轨上，提供螺丝压接或弹簧夹持的接线端子，用于集中连接和分配导线。是电控柜内最基础的布线元件。

```text
  导线 A ── 端子排 ── 导线 B（同端子号互通）
```

#### 主要特性

- **典型型号**：菲尼克斯 UTTB 2.5（2.5mm²）、魏德米勒 WDU 2.5
- **线径**：1.5mm² ~ 10mm²
- **类型**：直通端子、接地端子、熔断器端子、测试端子
- **标识**：可插标记条，标注线号
- **颜色**：灰色（普通）、黄绿（PE 接地）、蓝色（N 零线）

#### 典型接线规范

```text
  线号标识：每根线两端贴号码管
  色标规范：棕色=+24V、蓝色=0V、黄绿=PE
  扭矩：按端子规格拧紧（2.5mm² 约 0.5~0.6 N·m）
```

#### 应用场景

- 开关电源到各回路的分配
- PLC I/O 端子转接
- 传感器、阀门线缆汇集
- 所有电控柜的标准配置

---

### 6.2 拖链与工业线缆

![易格斯拖链与线缆](/img/in-post/electrical-components/cable-drag-chain.png)

#### 工作原理

拖链（能量链）保护运动部位电缆反复弯曲时不被拉扯、磨损或缠绕。内部电缆随机械运动弯曲，拖链承载弯曲半径和导向。

```text
  固定端 ── [拖链弯曲段] ── 移动端（龙门 / 机械手）
  电缆在拖链内有序排列，弯曲半径 ≥ 拖链允许值
```

#### 主要特性

- **典型品牌**：易格斯 E2/10 系列、怡合达拖链
- **类型**：桥式（上盖可开）、半封闭、全封闭
- **内宽**：10mm ~ 数百 mm
- **线缆要求**：使用**耐弯曲柔性电缆**（非普通固定敷设电缆）
- **填充率**：电缆占拖链截面 ≤ 60%

#### 典型选型

```text
  动力电缆：  伺服电机动力线（带屏蔽）
  编码器线：  专用编码器电缆（双绞屏蔽）
  信号线：    传感器电缆 / IO 线缆
  气管：      拖链内可同时走气管（需固定）
```

#### 应用场景

- 龙门 X/Y 轴运动部位
- 机械手关节线缆保护
- 升降平台电缆导向
- 任何往复运动部位的线缆管理

---

### 6.3 航空插头 / M12 连接器

![M12 工业连接器](/img/in-post/electrical-components/connector-m12.png)

#### 工作原理

工业连接器提供可插拔的防水电气连接，用于传感器、阀门等现场设备与电控柜之间的快速拆装。M12 是最常用的传感器连接器标准。

```text
  传感器 ── M12 插头 ── 电缆 ── M12 插座 ── 端子排 / PLC
  拔插即可维护，无需重新接线
```

#### 主要特性

- **典型品牌**：Lumberg、Binder、费斯托
- **接口**：M8（小型）、M12（标准）、M23（大电流）
- **针数**：3针（电源+信号）、4针、5针、8针
- **防护**：IP67（防水防尘）
- **接线**：焊接、螺丝、弹簧压接

#### 典型接线（M12 4针传感器）

```text
  针1（棕色）── +24V
  针3（蓝色）── 0V
  针4（黑色）── 信号
  针2 ── 未用或第二信号（视型号）
```

#### 应用场景

- 现场传感器快速连接
- 电磁阀插头（DIN 43650 或 M12）
- 设备模块化拆装（整机运输时断开）
- 潮湿、油污环境的可靠连接

### 6.4 IO 端子模块

![菲尼克斯 IO 端子模块](/img/in-post/electrical-components/io-terminal-module.png)

#### 工作原理

IO 端子模块（PLC-INTERFACE / IO 接线端子）是介于**现场线缆**与 **PLC IO 模块**之间的专用转接组件。现场传感器、电磁阀线缆先接到 IO 端子模块的弹簧端子上，再通过预制电缆或排线插头连接到 PLC 输入/输出模块，实现规范化、模块化接线。

```text
  现场传感器/阀门 ── 线缆 ──► [IO 端子模块] ── 预制电缆 ──► [PLC IO 模块]
                                    │
                                    └── 每点独立端子，带 LED 状态指示
```

与 6.1 节普通端子排的区别：IO 端子模块专为 PLC IO 回路设计，通常带**LED 通道指示**、**弹簧快速接线**和**与 PLC 模块一一对应的预制连接器**，拆装 PLC 模块时无需逐根拆线。

#### 主要特性

- **典型型号**：菲尼克斯 PLC-INTERFACE 系列、魏德米勒 PACTRON、西门子 SIMATIC TOP 连接
- **接线方式**：弹簧夹持（免螺丝）或螺钉压接
- **指示**：每路 DI/DO 带 LED，现场即可判断信号有无
- **配对**：按 PLC 模块型号选型（如 16 点 DI 对应 16 路端子模块）
- **标记**：自带标记区域或标签条，便于与电气图纸线号对应

#### 典型接线

```text
  现场侧（端子模块）：
    棕色 ──── +24V 公共端（DI 公共 / DO 公共）
    蓝色 ──── 0V 公共端
    黑色 ──── 各通道信号线（传感器输出 / 电磁阀线圈）

  PLC 侧：
    预制电缆插头 ──── 直接插入 PLC SM 模块前连接器
    无需在 PLC 端子逐根接线
```

#### 应用场景

- 标准电控柜规范化接线（图纸与实物一一对应）
- PLC 模块更换时快速插拔，不拆现场线
- 调试阶段通过 LED 快速判断某路 IO 是否有信号
- 与 3.7 远程 IO 模块配合：远程站内部同样使用 IO 端子模块整理现场接线
